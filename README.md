## 📌 Project Overview
This project is a **Linde-inspired industrial supply chain analytics case study** built using **Python, MySQL, and Power BI**.

The objective was to analyze:

- Inventory shortages
- Supply chain delivery performance
- Manufacturing utilization
- Forecast accuracy
- Product revenue analysis
- Customer priority insights

This simulates a real industrial analytics environment similar to **Linde operations**.

## 🧹 Data Cleaning & Feature Engineering
Performed using Python:

- Handled missing values
- Converted numeric text columns
- Converted date columns
- Created `Delivery_Delay_Days`
- Created `Inventory_Status`
- Created `Forecast_Accuracy_pct`
- Created `Expiry_Risk`
- Created `Estimated_Profit_USD`
